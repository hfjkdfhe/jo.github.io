08 LAUNCH

GitHub Pages 上架流程

Step 1
建立 GitHub Repository，例如：trivago-travel-site。
Step 2
把本頁 HTML 存成 index.html，上傳到 Repository。
Step 3
進入 Settings → Pages，選擇 Deploy from a branch。
Step 4
Branch 選 main，Folder 選 /root，按 Save。
Step 5
取得網址：https://你的帳號.github.io/trivago-travel-site/
